#Projeto SA-jogo descubra a palavra

##curso
Técnico em desinvolvimento de sistemas
UC: Lógica de programação

#Descrição de projeto
Desinvolvimento de um **Jogo de advinhação de palavras para web. O objetivo é aplicar conceitas fundamentais de lóica de programação, manifestação de
Dom e organização de código em um projeto real de desinvolvimento. O jogador deve descobrir uma palavra secreta tentando adivinhar letras, com um número limitado de tentativas.
